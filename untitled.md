# Markdown View에 오신 것을 환영합니다! 🎉

좌측에서 **마크다운**을 수정하면 우측에 *실시간*으로 반영됩니다.

## 주요 기능

- 📂 파일 열기 / 💾 저장 (`Ctrl+O` / `Ctrl+S`)
- 🔄 좌우 **동기 스크롤**
- 🌐 미리보기 그대로 **HTML 내보내기**

> 인용문은 이렇게 표현됩니다.

| 기능 | 단축키 |
|------|--------|
| 저장 | Ctrl+S |
| 열기 | Ctrl+O |
